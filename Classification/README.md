* 實作一個線性二元分類器，來根據人們的個人資料，判斷其年收入是否高於 50,000 美元
* X_train, Y_train, X_test

  * X_train, X_test : 每一列包含一個510維的資料作為一個樣本
  * Y_train: label = 0 代表  “<= 50K” 、 label = 1 代表  “ >50K ”
